# ALINX Board
Some fpga and software logic to test out the new ALINX AXU2CG-E development board.

## Things Tested
    - Xilinx XCZU2CG-1SFVC784E MPSOC device
    - AlINX AL321 USB debug/download cable
    - fan PWM control
    - PL LED
    - eMMC drive
    - QSPI boot
    - SD card boot
    - PS Ethernet
    - USB 3.0 hub

## Tings Not Yet Tested
    - CAN Bus
    - Camera Interface
    - Display Interface
    - Mini-HDMI Interface
    - PS LED
    - PL Ethernet
